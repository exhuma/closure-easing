Help

This page contains two demos. For any demo, you can select an easing function from the drop-down menu at the top of the page. During easing, the values are plotted to give you an idea what's happening. It shows the speed in function of position/time.

Additionally, the values are loged in the LOG tab. The logger displays the raw data values (rounded to 3 decimal places).

Position Easing
This demonstrates the easing functions by repositioning an element on the page. Simply click into the rectangle to re-position the element.

Color Easing
This demonstrates changing the color using the easing functions. Simply click on a color-swatch to start the animation.
